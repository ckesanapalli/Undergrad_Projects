Image analysis using PCA : 

Problem: 
The Yale face database contains 165 GIF images of 15 subjects (11 per subject). For each subject use 10 of the images for training and the remaining for testing. Develop a classifier using PCA on the training set images and use it to identify the image left out.

Result:
Subject 1, 4, 5, 6, 7, 9, 11, 14 are recognized correctly and rest are mismatched.
